# Library Management System

A full-stack Library Management System built using React, Spring Boot, MyBatis and SQLite.

## Features

- Add Books
- Edit Books
- Delete Books
- Search Books
- Pagination
- Dashboard Analytics
- Toast Notifications
- Loading Spinner
- Edit Modal
- Delete Confirmation Modal

## Tech Stack

### Frontend
- React
- Axios
- React Toastify

### Backend
- Spring Boot
- MyBatis
- SQLite
